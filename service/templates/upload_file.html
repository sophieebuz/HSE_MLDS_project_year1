Загрузите файл для получения предсказания

Прикрепленный файл должен быть в формате csv и содержать следующие поля: date,url,topic,tags,title,text

Пояснение полей:

  • date - дата публикации статьи
  • url - ссылка на новость
  • topic - тема статьи
  • tags - "подтема" статьи (более узкая, чем предыдущая колонка)
  • title - название статьи
  • text - текст статьи
